Project Info

About ShipIQ

Portfolio Coursework 2 — Option 2: Real-World Data Analysis & Business Problem Solving

Business Problem

An international e-commerce electronics company is experiencing a ~60% late delivery rate, causing increased customer complaints, poor ratings, and revenue loss. ShipIQ predicts delivery delays before dispatch so the company can take proactive action.

Dataset

E-Commerce Shipping Dataset sourced from Kaggle (by Prachi13). Contains 10,999 real shipment records with 12 features including warehouse block, shipment mode, product details, and delivery outcome.

ML Model

Random Forest Classifier trained on 80% of the dataset and tested on 20%. Achieved ~69% accuracy with strong precision and recall scores. Feature importance analysis revealed discount and weight as top predictors.

Tech Stack

Built with Python for data analysis and machine learning, Flask for the backend API, and vanilla HTML/CSS/JS for the frontend dashboard.

Python Flask Pandas Scikit-learn Random Forest K-Means Azure GitHub

Project Structure

Well-structured Git repository with separate notebooks for ETL, EDA, Clustering, and Modelling. All outputs, charts, and documentation maintained in organised folders.

Cloud Architecture

Azure Blob Storage for raw data, Azure SQL for structured storage, Azure VM for computation. Model deployment via Azure ML REST API endpoint for real-time predictions.
